Audible Maps Wayfinding

Project Goal

The goal of this project was to create a user-friendly, accessible navigation system to help visually impaired individuals find their way around the ASU campus. The prototype, developed using HTML, provides specific directions when a button is clicked, utilizing audible feedback to guide users. This project aimed to enhance independence for visually impaired students and visitors, ensuring they can navigate campus with confidence.
